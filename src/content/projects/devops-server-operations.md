## Context

Beyond application code, I regularly work close to the systems that build, ship, and run software. That includes CI/CD pipelines, container-based deployments, Linux servers, and the shell tooling around them.

## Focus

- Maintaining GitLab and GitHub CI/CD workflows.
- Working with Docker-based build and runtime setups.
- Administering Linux servers and diagnosing operational issues.
- Automating repeated tasks with shell-first workflows and AI-assisted command-line support.

## Why it matters

Reliable software delivery depends on more than code. Good operational work reduces friction, makes failures easier to understand, and keeps systems maintainable over time.
